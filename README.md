# Prodesk IT - Sprint 2: Cash-Flow Tracker

A dynamic, high-performance Expense Tracker web application built for the Sprint 2 engineering requirements. This project features real-time mathematical calculations, dynamic DOM manipulation, and a sleek dark-mode aesthetic without using external frontend frameworks.

## 🛠️ Tech Stack
- HTML5
- Vanilla CSS3 (Grid Architecture & Custom Styling)
- Vanilla JavaScript (ES6)

## ✨ Features Completed
- **Real-Time Calculations:** Instantly deducts logged expenses from the set total salary and updates the remaining balance dynamically.
- **Dynamic DOM Manipulation:** Automatically generates expense list items (`<li>`) and interactive delete buttons directly from JavaScript.
- **Reversible Logic:** Features a fully functional "Delete" button that removes specific expenses from the DOM and perfectly recalculates the total expenses and remaining balance.
- **Data Validation:** Prevents form submission and triggers UI error messages if input fields are left blank or if negative values are entered.
- **Modern UI Architecture:** Implemented a sleek dark-mode aesthetic featuring a vertically and horizontally centered layout using CSS Grid, complete with custom neon borders (`#00e5ff`). 
- **State Protection:** Automatically disables the "Total Salary" input after the initial entry to prevent accidental user overrides during calculation.
